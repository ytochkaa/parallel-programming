## 6. Лабораторные задания по теме: Вычисление двойных интегралов (метод Монте-Карло)

### Задание
Написать параллельную MPI программу приближенного вычисления двойного интеграла методом Монте-Карло с большой точностью. Число случайных точек взять равным n = 10^7 или больше.

Функция: f(x,y) = exp((x+y)^2)

Область интегрирования: D = { 0 < x < 1, 0 < y < 1-x } (треугольник). Площадь области S = 0.5.

Для контроля точности приближенного вычисления использовать расчет в системе MathCad или аналогичной. Оценить ускорение и эффективность параллельной программы, подготовить отчет.

### Метод решения
Оценка интеграла методом Монте-Карло:
I = S * (1/N) * sum_{i=1}^{N} f(x_i, y_i),
где (x_i, y_i) - случайные точки, равномерно распределенные в D.

Генерация случайной точки внутри области:
- x = случайное число из [0, 1)
- y = случайное число из [0, 1-x)   (равномерно на отрезке)

### Параллельная реализация (MPI)
- Распределение общего количества точек N между MPI-процессами (поровну, остаток отдается первым процессам).
- Каждый процесс генерирует свои точки, вычисляет локальную сумму значений f(x,y).
- С помощью MPI_Reduce все локальные суммы суммируются на процессе 0.
- Процесс 0 вычисляет итоговое приближение интеграла и выводит результат.

### Оценка ускорения и эффективности
- Измерить время выполнения параллельной части для разного числа процессов p = 1, 2, 4, 8, ...
- Ускорение: S(p) = T1 / Tp, где T1 - время на 1 процессе, Tp - время на p процессах.
- Эффективность: E(p) = S(p) / p.

### Контроль точности
Сравнить полученное значение с эталонным, вычисленным в MathCad или с помощью функции dblquad из библиотеки SciPy в Python. Оценить абсолютную и относительную погрешность.
