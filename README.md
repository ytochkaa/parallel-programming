## 6. Лабораторные задания по теме: Вычисление двойных интегралов (метод Монте-Карло)

### Задание
Написать параллельную MPI программу приближенного вычисления двойного интеграла методом Монте-Карло:

$$
\iint_D f(x,y)\,dx\,dy
$$

с большой точностью. Число случайных точек взять равным \( n = 10^7 \) или больше.  
Вид функции \( f(x,y) \) и область интегрирования:

$$
f(x,y) = e^{(x+y)^2}, \quad D = \{0 < x < 1,\; 0 < y < 1-x\}.
$$

Для контроля точности приближенного вычисления использовать расчет в системе MathCad или аналогичной.  
Оценить ускорение и эффективность параллельной программы, подготовить отчет.

**Метод решения** – метод Монте-Карло с равномерным распределением точек в области \( D \).  
Площадь области \( S = \frac{1}{2} \). Оценка интеграла:

$$
I \approx S \cdot \frac{1}{N} \sum_{i=1}^{N} f(x_i, y_i),
$$

где \( (x_i, y_i) \) – случайные точки в \( D \).  
Генерация точки внутри области: \( x \in [0,1) \) равномерно, затем \( y \in [0, 1-x) \) равномерно.

**Параллельная реализация (MPI):**
- Распределение количества точек между процессами.
- Каждый процесс вычисляет локальную сумму значений функции.
- Глобальная сумма собирается на процессе 0 с помощью `MPI_Reduce`.
- Процесс 0 вычисляет итоговое приближение интеграла и выводит результат.

**Оценка ускорения и эффективности:**  
Измерить время выполнения параллельной части для разного числа процессов \( p = 1,2,4,8,\dots \).  
Вычислить ускорение \( S(p) = T_1 / T_p \) и эффективность \( E(p) = S(p) / p \).

**Контроль точности:**  
Сравнить полученное значение с эталоном, вычисленным в MathCad или через `scipy.integrate.dblquad` в Python.
