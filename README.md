## 1. Лабораторные задания по теме: Двухточечные обмены

### Задание 4
На процессе с номером 0 заполнить массив `x_global[]` случайными числами из диапазона `[-9, 9]`.  
Размер массива задать равным `n = (size + 1) * size / 2`, где `size` — число используемых процессов.

Разослать вектор `x_global[]` по процессам следующим образом:
- на 1-й процесс — нулевой элемент массива `x_global[]`
- на 2-й процесс — первый и второй элементы массива `x_global[]`
- и так далее по аналогии

## 2. Лабораторные задания по теме: Коллективные операции

### Задание 8
На процессе с номером 0 заполнить матрицу `A_global[n][m]` случайными числами из диапазона `[-9, 9]`.  
Размер матрицы задать равным `n = 4 * size`, `m = 5`, где `size` — число используемых процессов.

Распределить матрицу `A_global[][]` по процессам.  
На каждом процессе найти минимальный элемент в распределенной части матрицы и вычислить сумму минимальных элементов с каждого процесса.  

Выдать результат на процессе с номером `size - 1`.

## 3. Лабораторные задания по теме: Распараллеливание матричных алгоритмов

Матрицы заполнить случайными числами из диапазона `[-9, 9]`.  
Размер матриц задать равным `n = 4`, `m = 4 * size`, где `size` — число используемых процессов.

Результат получить на каждом процессе.

## 4. Лабораторные задания по теме: Алгоритмы суммирования

### Задание 3
Написать параллельную MPI программу вычисления скалярного произведения двух векторов `X` и `Y` длиной `n = 10^6 – 10^7`.  
Для удобства отладки оба вектора состоят из единиц.

Требуется распределить векторы между процессами, а затем, в зависимости от варианта, реализовать алгоритм суммирования.

Формула скалярного произведения векторов:

$$S = \sum_{i=1}^{n} (x_i \cdot y_i)$$

**Вариант 3:** Модифицированная каскадная схема суммирования (МКС).
