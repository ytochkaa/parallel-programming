## 4. Лабораторные задания по теме: Алгоритмы суммирования

### Задание 3
Написать параллельную MPI программу вычисления скалярного произведения двух векторов `X` и `Y` длиной `n = 10^6 – 10^7`.  
Для удобства отладки оба вектора состоят из единиц.

Требуется распределить векторы между процессами, а затем, в зависимости от варианта, реализовать алгоритм суммирования.

Формула скалярного произведения векторов:

$$S = \sum_{i=1}^{n} (x_i \cdot y_i)$$

**Вариант 3:** Модифицированная каскадная схема суммирования (МКС).

**Реализация:** [lab_dot/](lab_dot/) – декомпозиция векторов, каскадное суммирование с чередованием.

## 5. Лабораторные задания по теме: Вычисление определенных интегралов

### Задание (Метод Симпсона)

Написать параллельную MPI программу приближенного вычисления определенного интеграла  

\[
\int_a^b f(x) dx
\]

с большой точностью. Число разбиений взять равным \( n = 10^7 \) или больше.  
Вид обобщенной формулы численного интегрирования – **метод Симпсона** (формула парабол).

Подынтегральная функция выбирается из вариантов 1–19 (например, \( \sin(x), x^2, e^{-x}, \frac{1}{1+x^2} \) и др.).  
Для контроля точности используется расчет в MathCad или аналогичном ПО.  
Необходимо оценить **ускорение** и **эффективность** параллельной программы.

#### Особенности реализации

- **Метод**: составная формула Симпсона (требует четного числа разбиений \(n\)).
- **Параллельная схема**: отрезок \([a,b]\) разбивается на \(n\) равных частей, каждый процесс получает непрерывный блок подотрезков (кроме, возможно, корневого, который также участвует в вычислениях).  
- **Коммуникация**: после локальных вычислений все частичные суммы собираются на корневом процессе с помощью `MPI_Reduce`.
- **Точность**: двойная точность (\(double\)), при \(n=10^7\) и методе Симпсона погрешность составляет порядка \(10^{-12}\)–\(10^{-14}\) для гладких функций.
