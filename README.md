## 4. Лабораторные задания по теме: Алгоритмы суммирования

### Задание 3
Написать параллельную MPI программу вычисления скалярного произведения двух векторов `X` и `Y` длиной `n = 10^6 – 10^7`.  
Для удобства отладки оба вектора состоят из единиц.

Требуется распределить векторы между процессами, а затем, в зависимости от варианта, реализовать алгоритм суммирования.

Формула скалярного произведения векторов:

$$S = \sum_{i=1}^{n} (x_i \cdot y_i)$$

**Вариант 3:** Модифицированная каскадная схема суммирования (МКС).
